### 🌌 About Me

I am a highly motivated **Fullstack Web Developer** and **AI Model Developer & Trainer**. My expertise lies in designing scalable web architectures and building specialized AI/ML pipelines (from custom dataset preparation to production deployments). I thrive on creating robust backend microservices, highly interactive modern UIs, and deep learning setups that bridge user experiences with cognitive models.

---

### 🚀 Detailed Area of Work

#### 🎨 Frontend Engineering
* **Responsive Layouts & Aesthetics:** I design custom responsive systems using modern vanilla CSS structures, TailwindCSS, and neomorphic/glassmorphic elements.
* **Component-Driven Apps:** Experienced in building single-page and server-side rendered applications utilizing **React** and **Next.js** for maximum performance and SEO.
* **State Management & Interactions:** Implementing state logic using Redux, Context API, and real-time custom listeners.

#### ⚙️ Backend Engineering
* **Fast & Scalable APIs:** Building secure RESTful and GraphQL endpoints using **Node.js (Express)**, **Python (FastAPI, Flask)**, and **Java (Spring Boot)**.
* **Real-time Services:** Developing instant communication pipelines with **Websockets (Socket.io)** for chats, dashboard updates, and training monitors.
* **Database & Cache Systems:** Proficient in designing schemas, query optimization, and connection pooling with **PostgreSQL**, **MySQL**, **MongoDB**, and cache setups with **Redis**.

#### 🤖 AI Model Development & Training
* **Deep Neural Networks:** Training custom Convolutional Neural Networks (CNNs) for Computer Vision and sequential models for NLP using **PyTorch** and **TensorFlow**.
* **LLM Fine-Tuning:** Specializing in optimizing open-source models using parameter-efficient fine-tuning (LoRA, QLoRA) on domain-specific dataset distributions.
* **Data Pipelines & Curation:** Processing, cleaning, and custom labeling raw datasets to prepare high-quality training sets for machine learning models.
